# Atalhos de Teclado

O Aegis suporta diversos atalhos de teclado para acelerar seu fluxo de trabalho.

## Geral
- `I`: Ir para a Caixa de Entrada.
- `D`: Ir para o Dashboard.
- `S`: Ir para Configurações.
- `C`: Abrir o modal "Reportar Problema".
- `Ctrl + K`: Abrir o modal de "Reportar Problema" (alternativo).
- `Esc`: Fechar modais abertos ou limpar seleções.

## Inbox
- `J`: Mover para o próximo ticket.
- `K`: Mover para o ticket anterior.
- `Enter`: Abrir ticket selecionado.
- `R`: Atualizar a lista de tickets.

## Ações em Massa (com tickets selecionados)
- `A`: Abrir menu de atribuição.
- `S`: Abrir menu de status.
- `P`: Abrir menu de prioridade.
- `Esc`: Limpar seleção.
